# Mahjong Android v1.0.0

发布日期：2026-04-11

## 版本亮点

- 首个公开版本发布。
- 提供对局记分、统计、玩家管理、设置等核心能力。
- 基于 Kotlin + Jetpack Compose 构建，支持中英文资源管理。

## 安装包

- 文件名：`mahjong-v1.0.0-release.apk`
- 构建类型：`release`
- 签名方式：`release keystore`（本地 `keystore.properties` 配置）

## 已知说明

- 如首次安装失败，请确认设备允许安装未知来源应用。
- 若已安装旧版本且签名不同，请先卸载旧包后重装。
